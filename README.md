# 🚀 Harshitha K V V S S - Personal Portfolio

A sleek, responsive, and modern personal portfolio website built using **HTML5** and **CSS3**. This project showcases my technical skills, educational background, and recent software projects as an Information Technology student.

---

## 🌟 Features

* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Modern UI/UX:** Clean typography using *Playfair Display* and *DM Sans*.
* **Sticky Navigation:** Smooth user experience with easy access to all sections.
* **Organized Sections:** Includes About Me, Technical Skills, Projects, Education, and Contact info.
* **Performance:** Lightweight and fast-loading with zero external JavaScript dependencies.

---

## 🛠️ Tech Stack

* **HTML5:** Structural backbone of the site.
* **CSS3:** Custom styling including Flexbox, Grid, and Linear Gradients.
* **Google Fonts:** High-quality web typography.
